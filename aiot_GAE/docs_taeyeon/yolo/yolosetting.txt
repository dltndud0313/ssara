### 1. 필수 도구 설치 (에디터 및 pip)

파일을 수정할 `nano` 에디터와 파이썬 라이브러리 설치 도구인 `pip`가 없어서 설치했습니다.

Bash

```bash
apt-get update
apt-get install -y nano python3-pip
```

### 2. YOLO 라이브러리 설치

`ModuleNotFoundError: No module named 'ultralytics'` 에러 해결을 위해 설치했습니다.

Bash

```bash
pip3 install ultralytics
```

```bash
apt-get update && apt-get install -y python3-pip && pip3 install ultralytics
```

### 3. Numpy 버전 다운그레이드 (핵심 해결!)

`AttributeError: _ARRAY_API not found` 에러가 발생했습니다. 이는 최신 `ultralytics`가 `numpy 2.x` 버전을 설치했는데, ROS 2의 `cv_bridge`는 아직 `numpy 1.x` 버전만 지원하기 때문에 발생한 충돌입니다. 이를 해결하기 위해 버전을 낮췄습니다.

Bash

```bash
pip install "numpy<2"
```
